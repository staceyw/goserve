GoServe

Lightweight HTTP file server with WebDAV support in a single binary. Upload, download, edit, preview, and manage files from any browser.

Install

Linux / macOS
curl -fsSL https://raw.githubusercontent.com/staceyw/goserve/main/scripts/install.sh | bash
Windows (PowerShell)
iex (irm https://raw.githubusercontent.com/staceyw/goserve/main/scripts/install.ps1)

Downloads the binary into the current directory. Ready to run immediately.

Pick your platform binary.

Windowsx64 WindowsARM64 Linuxx64 LinuxARM64 / Pi macOSApple Silicon

Get Started

1

Download

Use the install script above or grab a binary for your platform.

2

Run

Serve the current directory:
./goserve or ./goserve -permlevel all

3

Browse

Open http://localhost:8080 in your browser.

Features

File Management

Upload files and folders, rename, delete, and edit text files with syntax highlighting — all from the browser.

WebDAV Server

Built-in WebDAV endpoint. Mount as a network drive on Windows, macOS, or Linux.

12 Themes

Catppuccin, Dracula, Nord, Solarized, Gruvbox, One Dark, Monokai, and more. Saved in your browser.

File Preview

Preview images, text, markdown, and code files directly in the browser without downloading.

Search & Filter

Real-time search with wildcard support. Use *.go, test*, or *config* patterns.

ZIP Download

Download entire directories as ZIP archives with one click.

Authentication

Optional HTTP Basic Auth with three permission levels: readonly, readwrite, and all.

GZIP Compression

Automatic response compression for faster transfers over slow networks.

Single Binary

All HTML, CSS, and JS embedded. ~8 MB, pure Go, no dependencies to install. Cross-platform.
